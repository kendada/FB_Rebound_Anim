#### Rebound是FaceBook出品的一个弹簧动画库，主要就是现有的动画离真实物理世界差别比较明显，为了让动画看起来真实自然，带有力量的效果，Rebound应运而生。两个比较重要的参数，一个是拉力Tension，一个是摩擦力Friction，拉力越大，弹簧效果越明显；摩擦力越大，弹框效果阻力越大、越不明显。
